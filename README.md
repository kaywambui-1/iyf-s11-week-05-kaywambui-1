## Project Description
This project demonstrates JavaScript DOM manipulation and event handling. It includes exercises on event listeners, keyboard events, event bubbling, event delegation, and form validation. The project was built to practice creating interactive web pages using HTML, CSS, and JavaScript.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Visual Studio Code
- Git
- GitHub

## Features
- Responds to mouse and keyboard events.
- Interactive task list where users can add, complete, and delete tasks.
- Contact form with real-time name and email validation.
- Prevents invalid form submissions and displays a success message.

## Lessons Learned
- How to select HTML elements using the DOM.
- How to use `addEventListener()` for different events.
- How keyboard events and the event object work.
- How event bubbling and event delegation improve efficiency.
- How to collect form data using `FormData`.

## Challenges Faced
One challenge was getting event delegation to work correctly in the task list. I solved it by checking `event.target` and ensuring clicks were handled by the parent element.
